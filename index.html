Usage Guidelines

Rules for consistent brand representation across all touchpoints.

Do

  • Use General Sans Regular (400) and Medium (500) only
  • Use pill-shaped buttons (24px border-radius) for CTAs
  • Use Primary Blue (#0E36A5) for interactive elements
  • Use Dark Blue (#030B21) for all body text
  • Maintain clear space equal to the icon height around the logo
  • Use country photos from the approved set
  • Keep line height at 1.4 (140%) across all text

Don't

  • Never use Bold weight — only Regular and Medium
  • Don't stretch, rotate, or recolor the logo
  • Don't place the logo on busy backgrounds without a container
  • Don't use gradients on buttons — solid Primary Blue only
  • Don't use colors outside the approved palette
  • Don't use stock photos — only approved country photography
  • Don't mix fonts — General Sans is the only typeface

API

Fetch design tokens programmatically. All assets support CORS and are cached on Vercel CDN.

// Fetch design tokens

// Use assets directly
